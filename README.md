# cmap

Chunkmap, a territory control mod for Forge/1.20.1



## Contributing

We ask that commit and PR names are as follows:
* `feat:` for new features
* `fix:` for bug fixes
* `refactor:` for code refactoring
* `docs:` for documentation changes
* `test:` for adding or updating tests
* `chore:` for other changes that don't modify src or test files
* `ci:` for changes to CI configuration files and scripts
* `revert:` for reverting a previous commit
* `WIP:` for work-in-progress commits
* `release:` for releasing a new version
* `deps:` for updating dependencies
* `build:` for changes that affect the build system or external dependencies
* `rec`: for editing records
